Algoritmo Fillpoly

Requisitos minimos:

  • O algoritmo a ser implementado deve usar aritmética incremental (requisito obrigatório), conforme orientações apresentadas em sala de aula;
  • Pode ser desenhando qualquer tipo de polígono (convexo, côncavo ou com autointerseção);
  • O usuário pode desenhar um ou mais polígonos de qualquer formato na tela do computador, usando para isso cliques do mouse;
  • Cada polígono pode ser selecionado individualmente pelo usuário, usando para isso cliques do mouse sobre a interface onde estão desenhados;
  • O usuário pode selecionar um polígono e trocar a cor de preenchimento (requisito obrigatório);
  • Definida a geometria do polígono e sua cor de preenchimento deve-se pintar apenas os pixels que pertençam à região interna do polígono (requisito obrigatório);
  • Deve estar disponível ao usuário a opção de pintar ou não pintar as arestas dos polígonos. Neste caso a cor das arestas será sempre fixa (em amarelo);
  • Polígonos selecionados podem ser excluídos da lista de polígonos manipuladas pelo software.

Como usar:

  1. Clicamos no botão "Desenhar"
  2. Construimos nosso polígono
    1. Para finalizar um polígono basta clicar no primeiro vertice
  3. Para preencher o polígono:
    1. Selecionamos o polígono na nossa lista de polígonos
    2. Escolhemos a cor desejada
    3. Então clicamos no botão "Preencher polígono"
  4. Para excluir um único polígono:
    1. Selecionamos o polígono na nossa lista de polígonos
    2. Então clicamos no botão "Excluir polígono"
  5. Podemos também limpar a tela com o botão "Limpar tela"
